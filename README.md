# Inventory Itemization
Inventory Itemization - Simple IoT Inventory Manager

## Fields
* Manufacture
* Model Number
* OS Version
* OS Status (Rooted / Root)
* Serial Number
* Last Known Location
    * Latitude
    * Longitude
    * Name
* Report In Timestamp
* Physical Condition
* IP Address
* MAC Address (Manufacture)

## Server Settings
* Frequency
    - Default is 24 hours (24, 12, 8, 6, 4, 2, 1)

## Client 

### Settings
* Location Name
* Server URL
* Frequency (non editable)

### View
* Information
* Settings
